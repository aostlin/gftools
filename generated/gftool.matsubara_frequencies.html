gftool.matsubara_frequencies¶

gftool.matsubara_frequencies(n_points, beta)¶

Return fermionic Matsubara frequencies \(iω_n\) for the points n_points.

Parameters
n_pointsint array_like

Points for which the Matsubara frequencies \(iω_n\) are returned.

betafloat

The inverse temperature \(beta = 1/k_B T\).

Returns
matsubara_frequenciescomplex np.ndarray

Array of the imaginary Matsubara frequencies

Examples

>>> gt.matsubara_frequencies(range(1024), beta=1)
array([0.+3.14159265e+00j, 0.+9.42477796e+00j, 0.+1.57079633e+01j, ...,
       0.+6.41827379e+03j, 0.+6.42455698e+03j, 0.+6.43084016e+03j])
